# Sucana Ad Image Generator

A Claude Code skill for generating ad creative images using AI. Produces platform-ready visuals for Meta, Google, LinkedIn, and TikTok campaigns.

## Install

Drop the skill folder into your `.claude/skills/` directory.

## Requirements

- Claude Code CLI
- Image generation MCP integration
